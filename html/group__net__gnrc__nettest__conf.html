    ◆ CONFIG_GNRC_NETTEST_MSG_QUEUE_SIZE_EXP

    #define CONFIG_GNRC_NETTEST_MSG_QUEUE_SIZE_EXP   (3U)

    Default message queue size to use for the nettest thread (as exponent of 2^n).

    As the queue size ALWAYS needs to be power of two, this option represents the exponent of 2^n, which will be used as the size of the queue.
