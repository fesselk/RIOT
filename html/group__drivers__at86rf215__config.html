    ◆ CONFIG_AT86RF215_TRIM_VAL

    #define CONFIG_AT86RF215_TRIM_VAL   (0)

    Trim value for the external crystal oscillator.

    Each increment adds 300nF capacitance between the crystal oscillator pins TCXO and XTAL2.

    Range: 0..15

    Use in conjunction with

    See also
    CONFIG_AT86RF215_USE_CLOCK_OUTPUT and a frequency meter connected to the clock output pin of the AT86RF215. Tweak the value until the measured clock output matches 26 MHz the best.

    ◆ CONFIG_AT86RF215_USE_CLOCK_OUTPUT

    #define CONFIG_AT86RF215_USE_CLOCK_OUTPUT

    Set to 1 if the clock output of the AT86RF215 is used as a clock source on the board.

    Otherwise it is turned off to save energy.
