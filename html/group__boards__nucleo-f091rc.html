    Support for the STM32 Nucleo-F091RC.

    Overview

    The Nucleo-F091RC is a board from ST's Nucleo family supporting a ARM Cortex-M0 STM32F091RC microcontroller with 32Kb of RAM and 256Kb of ROM.

    Flashing the device

    The ST Nucleo-F091RC board includes an on-board ST-LINK V2 programmer. The easiest way to program the board is to use OpenOCD. Once you have installed OpenOCD (look here for installation instructions), you can flash the board simply by typing

    make BOARD=nucleo-f091rc flash

    and debug via GDB by simply typing

    make BOARD=nucleo-f091rc debug

    Supported Toolchains

    For using the ST Nucleo-F091RC board we strongly recommend the usage of the GNU Tools for ARM Embedded Processors toolchain.
